# ⚙️ Portfolio Backend API

RESTful API backend for my personal portfolio website built with **Node.js, Express.js and MongoDB** following MVC Architecture.

## ✨ Features
- 📬 Contact form API
- 🗄️ MongoDB database integration
- 🌐 CORS enabled for frontend connection
- 🏗️ MVC Architecture (Models, Views, Controllers)
- 🛣️ Organized REST API routes
- 🔐 Environment variables support

## 🛠️ Tech Stack
- **Runtime:** Node.js
- **Framework:** Express.js
- **Database:** MongoDB
- **Architecture:** MVC Pattern

## 🚀 Run Locally
```bash
git clone https://github.com/MahakDalwani/portfolio_backend.git
cd portfolio_backend
npm install
npm start
```
